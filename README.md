# AnkeRead
  这个东西市面上有很多，但是要么广告特别多，影响阅读体验，
  要么就是书不全，要么就是打着免费的幌子变相收费。
  所以我想开发一款书全，真正无广告，免费的小说阅读器。

开发进度:
1. 主体框架完成
2. 热搜，换一批，搜索历史完成
3. 搜索功能完成
4. 书籍详情界面完成
5. 书架界面完成，书库界面完成
6. 分类完成
